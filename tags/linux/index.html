一些常用的Linux命令

MV命令—-重命名文件和移动文件

mv [-f | -i | -n][-v] source target 重命名文件
mv [-f | -i | -n][-v] source … dictionary 移动文件
-f: 强制覆盖文件，并且会覆盖之前的-i,-n选项
-i: 如果有重复文件的话，提示用户
-n: 如果已经存在文件，不覆盖，并且会覆盖之前的-i，-f选项
-v：可视化选项
