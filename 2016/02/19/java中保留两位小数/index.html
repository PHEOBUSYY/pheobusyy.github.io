如何避免科学计数法

在java中，如果浮点数大于999999.0的时候，会通过科学计数法莱显示浮点数，所以在编程的时候要注意 如何避免科学计数法，可以通过下面提供的方法来避免科学计数法

关于format的pattern详解

pattern转义符表格

符号 位置 含义
0 数字 数字
# 数字 数字，如果是0就不显示
. 数字 小数和整数分隔符
- 数字 负数
, 数字 整数分隔符
E 数字 科学计数法分隔符
; 分隔符 多个pattern之间分隔符
% 前缀后缀 百分位
‘ 前缀后缀 转义符号
\u2030 前缀后缀 千分位
\u00A4 前缀后缀 货币符号 
输出:
pattern=00.## ----->100000
pattern=##.00 ----->100000.00
pattern=,###.00 ----->100,000.00
pattern=,##.00 ----->10,00,00.00
pattern=0.00E0 ----->1.00E5
pattern=##% ----->10000000%
pattern=###,##0.00 ----->100,000.00
pattern=###‰ ----->100000001‰
pattern=###¤ ----->100000￥
pattern='#'#.## ----->#100000
pattern=''##.## ----->'100000
pattern=AAAA##.## ----->AAAA100000
