状态模式

状态模式是一种行为模式，可以理解为在面向对象中通过切换不同的状态来调用不同的逻辑。也就是说对象的行为取决于对象内部的不同状态
状态模式由三个角色组成，包括：抽象状态接口，具体状态实现，上下文对象。其中抽象接口是对行为的抽象定义，具体状态实现于抽象状态接口
最后在上下文中来实现状态转化和状态的方法调用。
